# AWS Infrastructure Automation with Terraform

## Overview
Automated AWS infrastructure provisioning using reusable 
Terraform modules, reducing deployment time by 40%.

## Infrastructure Components
- VPC with public and private subnets
- EKS cluster for Kubernetes workloads
- IAM roles and policies (least-privilege)
- S3 buckets with versioning
- Route tables and security groups

## Technologies Used
- Terraform (modules, remote state, workspaces)
- AWS (VPC, EKS, IAM, S3, EC2)
- Python & Bash automation scripts
- GitHub Actions CI/CD

## Key Achievements
- Reduced provisioning time by 40%
- Eliminated environment drift
- Enabled version-controlled deployments
- S3 + DynamoDB remote state management

## Skills Demonstrated
AWS | Terraform | Kubernetes | DevOps | IaC
